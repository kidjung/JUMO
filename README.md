# 프로젝트 소개

> 배포주소 http://jumo.ga

JUMO 프로젝트는 트레이딩 알고리즘을 이용한 종목 주가 예측과 모의투자를 제공하는 서비스입니다. OHLC데이터와 MACD알고리즘을 이용하여 단기 및 장기 추세를 알 수 있고, 사용자는 이를 참고하여 모의투자를 해볼 수 있는 기능을 구현하였습니다.
